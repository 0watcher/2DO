## About the project
Yet another simple old-school to-do list as single-executable with some additional features for companies with only one PC and no connection to the internet.

## Dependencies
- clang with C++20
- CMake
- SQLiteCpp
- fmt
- googletest
  
## Building
```
cd 2DO
git submodule update --recursive
mkdir build
cd build
cmake .
```
